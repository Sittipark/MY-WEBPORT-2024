About Me

สวัสดีครับ ผมนายสิทธิภาคย์ แก้วกา จุดเริ่มต้นของความสนใจด้านเทคโนโลยีของผมต้องย้อนกลับไปเมื่อสมัยที่ผมเรียนอยู่ในระดับชั้นมัธยมศึกษาปีที่ 2 ในวันนั้นเป็นวันแรกที่ผมได้ลองเขียนโปรแกรมภาษา Python โดยสิ่งที่ผมได้ลองเขียน ได้แก่ การคำนวณเกรดเฉลี่ย, การหาพื้นที่และปริมาตร เป็นต้น ซึ่งมันทำให้ผมรู้สึกชอบในการเขียนโปรแกรมมาก ๆ ผมรู้สึกสนุก รู้สึกได้ถึงความท้าทาย เมื่อเราต้องเผชิญปัญหา และสิ่งนี้เป็นจุดเริ่มต้นของการเรียนและศึกษาเพิ่มเติมด้วยตนเองอย่างจริงจังของผม ผมได้ลองทำอะไรหลาย ๆ อย่างไม่ว่าจะเป็นลองศึกษาภาษา Programming อื่น ๆ ที่ไม่ใช่ Python อย่าง C, C++ ได้ลองเขียนเว็บไซต์โดยใช้ HTML, CSS, Java Script ต่อยอดด้วยการลองใช้ Framework ต่าง ๆ จนผมได้มาพบเจอเข้ากับศาสตร์ของ AI & Data Science ซึ่งหลังจากที่ผมได้ลองเรียน ผมรู้สึกชื่นชอบมาก ๆ เพราะผมชอบที่เราสามารถนำ Data ต่าง ๆ มาวิเคราะห์แนวโน้ม แก้ปัญหา และหาคำตอบในสิ่งที่เราอยากรู้ได้ รวมถึง AI ที่สามารถเรียนรู้และพัฒนาได้อย่างไม่หยุดยั้ง ซึ่งเป็นสิ่งที่ท้าทายและน่าสนใจอย่างมาก นอกจากนี้ผมยังได้ลองสร้างโมเดล Machine Learning เพื่อทำนายผลลัพธ์ของสิ่งต่าง ๆ อีกด้วย ต่อจากนั้นมาผมก็พัฒนาตนเองตลอด ผมมองตัวเองเปรียบเสมือน น้ำไม่เต็มแก้ว ยังคงมีความรู้อีกหลายอย่างที่ผมยังไม่รู้ ผมเลยอยากจะพัฒนาตนเองไปเรื่อย ๆ เพื่อให้บรรลุถึงเป้าหมายของผมที่อยากทำงานในแวดวงเทคโนโลยี ปัญญาประดิษฐ์ และวิทยาการข้อมูลครับ
